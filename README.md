# superheroes-finder

## **User Story**

**WHEN** we you enter a Marvel SuperHero name in the form  
**THEN** the website returns that superheroes bio and then any movies they are in **// THIS this doesn't work as the API requires CORS** 

#### New Requirement
**WHEN** the page is loaded it displays the next marvel movie with an overview, poster and release date  

## **Requirements**

* 2 server-side API's  
* CSS framework other than Bootstrap  
* Be interactive (i.e., accept and respond to user input).  
* Do NOT use alerts  
* Mobile first design  
* clean repo  
* clean README
